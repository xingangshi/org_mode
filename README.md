## 2. [Org mode 快捷键](/org_mode_shortcuts.md)

### 2.1 列视图

使用 `C-c C-x C-c` 进入列模式。

### 2.2 日历视图

使用 `C-c a a` 进入日历视图。

### 2.3 切换日期或时间的自定义编辑和显示

使用 `C-c C-x C-t` 切换日期/时间的自定义格式显示
